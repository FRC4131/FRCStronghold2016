Nav6 LabVIEW Library

The following Vis are included in the Nav6 LabVIEW Library. API details are available for each VI by clicking on the link.

The library is structured similarly to the FIRST Libraries (open, close, get/set refnum, etc). It does use a parallel loop for reading the serial port, which is simlar to the compressor (which uses a parallel loop that the user puts in Periodic Tasks).
